# Contexta / 译境

译境是一款面向上下文和对话场景的 AI 翻译工具。它不只处理一句话的直译，也关注说话人身份、语气、背景信息和连续对话里的上下文，让译文更接近真实交流。

## 功能

- 单轮翻译：输入原文、目标语言、说话人身份、语气风格和可选背景信息，生成更贴合场景的译文。
- 多轮翻译：为双方创建对话会话，按上下文连续翻译消息，适合模拟聊天、邮件往来和跨语言沟通。
- 翻译审计：对译文进行质量评估，给出可选的修改建议，并支持基于建议重新润色。
- 历史记录：保存过往翻译，便于回看和复用。
- 请求日志：记录请求、响应和 cURL 命令，方便排查模型接口问题。
- 模型配置：支持 OpenAI 兼容接口，可管理多个 API / 模型预设，并可为审计等任务指定独立配置。
- 提示词设置：内置单轮翻译和对话翻译的系统提示词模板，可按使用习惯调整。

## 注意事项

- 审计模型仅支持结构化输出的 OpenAI 兼容接口。

## 开发

```bash
pnpm install
pnpm tauri dev
```

## 版本同步

手动修改 `package.json` 中的版本号后，运行：

```bash
pnpm sync-version
```

该命令会同步更新 Tauri 配置中的版本号。
